history Architecture Evolution

looks_one Phase 1 — Node.js + MySQL 30 days
Express API server with MySQL database, REST endpoints, server-side authentication. Replaced due to hosting complexity.
looks_two Phase 2 — Firebase Firestore 35 days
Firestore NoSQL database with real-time listeners. Bulk import scripts in Node.js. Migrated to eliminate server maintenance.
looks_3 Phase 3 — Google Sheets + Apps Script (Current) 13 days
Zero-infrastructure backend: Google Sheets as database, Apps Script as API (JSONP). Maximum simplicity for maintenance. Offline fallback via localStorage cache.
